Quickstart Guide

Integrating Ably is simple. In this guide we’ll run you through the basics.

Whilst our client libraries do support every popular language and platform, for simplicity we have used Javascript as the primary language for this guide.

Adding the Ably realtime library

Simply include the Ably Javascript library within the <head> of your HTML page as follows:

<script src="http://cdn.ably.io/lib/ably.js"></script>

The basics: Receiving a message

To connect to Ably, you need an API key. In this example we have provided a temporary API key, however if you are signed up and visit your application dashboard, you will be able to use your own API key. If you do not have an API key yet, why not sign up now for a free app and API key.

var app = new Ably.Realtime('OPk-Mg.FpJlvg:wr_DvMODwpUXwrAc');
app.on('connect', function() {
  alert("That was simple, we're now connected to Ably in realtime");
});

Messages are broadcasted on channels. Next we will subscribe to a channel and listen for messages that are sent to us from other connected clients or servers

var channel = app.channel('quickstart');
channel.on('greeting', function(data) {
  alert("Received a greeting message - " + data);
});

The basics: Sending a message

Publishing a message to a channel is extremely straightforward. As this is commonly performed by both connected devices and servers, here are examples in multiple languages

var channel = app.channel('quickstart');
channel.publish('greeting','hello!');
Channel channel = app.channel('quickstart');
channel.publish('greeting','hello!');
Channel channel = app.channel('quickstart');
channel.publish('greeting','hello!');
channel = app.channel('quickstart')
channel.publish('greeting','hello!')
channel = app.channel('quickstart')
channel.publish 'greeting', 'hello!'
$channel = $app.channel('quickstart');
$channel('quickstart').publish('greeting', 'hello!');
-(Channel) channel = app.channel('quickstart');
channel('quickstart').publish('greeting','hello!');

Sending a message via the REST API

We recommend using our REST client libraries over direct REST API access. However, if you want to comunnicate directly over HTTP with Ably it’s as simple as:

curl https://rest.ably.io/channels/quickstart/publish \
  --user OPk-Mg.FpJlvg:wr_DvMODwpUXwrAc \
  --data "name=alert&payload=Hello"

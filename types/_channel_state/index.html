blang[ruby].
Ably::Realtime::Channel::STATE is an enum-like value representing all the Realtime Channel states. STATE can be represented interchangeably as either symbols or constants.

h4. Symbol states ```[ruby] :initialized # => 0 :attaching # => 1 :attached # => 2 :detaching # => 3 :detached # => 4 :failed # => 5 ``` h4. Constant states ```[ruby] Connection::STATE.Initialized # => 0 Connection::STATE.Attaching # => 1 Connection::STATE.Attached # => 2 Connection::STATE.Detaching # => 3 Connection::STATE.Detached # => 4 Connection::STATE.Failed # => 5 ``` h4. Example usage ```[ruby]
  1. Example with symbols
    channel.on(:attached) { … }
  1. Example with constants
    channel.on(Ably::Realtime::Channel::STATE.Attached) { … }
  1. Interchangeable
    Ably::Realtime::Channel::STATE.Attached == :attached # => true
    ```

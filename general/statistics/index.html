The Ably system can be queried to obtain usage statistics for a given application, and results are provided aggregated across all channels in use in the application in the specified period. Stats may be used to track usage against account quotas. The details on how to retrieve statistics are available in the REST API documentation, Real-time client library statistics documentation and REST client-library statistics documentation.

Example request:

curl https://rest.ably.io/stats?unit=hour \
 -u "{{API_KEY}}"

REST Client Library

The Ably REST client library is available in most popular languages including Javascript, Java, Python, Ruby, .NET, PHP, Node.js, iOS and more…

Whilst you can interact directly with our REST API, we advocate the use of our REST client libraries as they strictly follow our REST API specification, they provide convenience, simplicity, and a number of additional features such as encryption and automatic data center fail-over. In addition, we have ensured that our client libraries provide a consistent API across all languages.

Primary interfaces

The client library exposes the Ably service with the following interfaces and methods:

  • Constructor & Usage
  • Channels & Messages
  • Presence
  • Authentication
  • History
  • Statistics
  • Encryption

REST API Specification

The Ably REST API can also be accessed directly over HTTPS.

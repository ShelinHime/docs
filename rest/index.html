REST Client Library API

The Ably REST client libraries provide a convenient and simple API to interact directly with the Ably REST API. All official client library APIs are consistent across every language, enable publishing and retrieving of messages, handle both basic and token authentication, work around unavailable data centres, include optional symmetrical encryption and provide access to presence and metrics history.

The Ably REST client library is available in most popular languages and platforms including including Javascript browser library, Java, .NET, Node.js, iOS, Python, Ruby, Go and more…

Download one of our REST client libraries now »

Dive into the documentation

The REST Client Library documentation API is broken down as follows:

  • Constructor & usage examples
  • Channels & Messages
  • Presence
  • Authentication
  • History
  • Statistics
  • Encryption

Use the Ably REST client libraries as they provide features that the REST API cannot support such as encryption or automatic fail-over to alternate data centers should the closest data center become unavailable.

REST API Specification

Whilst we recommend that you use one of the officially available Ably REST client libraries, you can accees the Ably REST API directly from any HTTP client.
